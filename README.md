## LaTex
Download all the LaTex packages. 

On GNU/Linux
On Debian-based distros (Ubuntu/Linux Mint):  
```shell
$ sudo apt-get install texlive-full
```
For Arch-based distros (Manjaro, Parabola, Antergos):
```shell
$ sudo pacman -S texlive-most texlive-lang
```
